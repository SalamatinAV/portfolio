Название приложения: Словарь

Технология: Приложение разработано с использованием Angular.

Основные характеристики приложения:

  • Приложение "Словарь" предназначено для изучения английских слов. Все слова хранятся на устройстве пользователя. Приложение состоит из трех вкладок: "Изучение", "Словарь" и "Избранные".
  • Вкладка "Изучение" предназначена для изучения слов. Слова можно изучать как с английского на русский, так и с русского на английский. Также можно выбрать случайный порядок слов и повтор слов. Пользователь может использовать : "Весь словарь", "Избранные слова", "Работа над ошибками" и "Выбрать последние". Слова, в которых были допущены ошибки, автоматически добавляются в словарь "Работа над ошибками".
  • Вкладка "Словарь" предназначена для добавления слов и отображает "Весь словарь". Также здесь пользователь может выполнять поиск в словаре, добавлять слова в избранные, редактировать словарь, удалить выбранное слово либо удалить весь словарь полностью.
