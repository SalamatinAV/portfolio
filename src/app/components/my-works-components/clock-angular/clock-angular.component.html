Название приложения: Стрелочные Часы

Технология: Приложение разработано с использованием Angular.

Основные характеристики приложения:

  • Приложение представляет собой классические стрелочные часы, отображающие текущее время. Дизайн часов включает в себя часовую, минутную и секундную стрелки.
  • Звуковые эффекты:
    • По умолчанию звук отключен. Однако, при включении звука, пользователь услышит звуковой сигнал, синхронизированный с движением секундной стрелки.
    • Когда часы пробивают полный час, приложение воспроизводит звук курантов, количество ударов которых соответствует текущему часу. Это добавляет атмосферу старинных часов.
  • Будильник:
    • Пользователь может устанавливать будильник, который будет срабатывать в заданное время, независимо от того, включен звук или нет.
    • Для настройки будильника необходимо навести указатель на часы и вращать колесико вверх или вниз, чтобы установить желаемое время срабатывания будильника.
    • После установки времени будильника, пользователь должен нажать на соответствующую кнопку для его активации.
    • Для деактивации будильника нужно произвести действие колесиком мыши, либо нажав на кнопку. Если же эти действия не были произведены, будильник отключится через две минуты автоматически.
