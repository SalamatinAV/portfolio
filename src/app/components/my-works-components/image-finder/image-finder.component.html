Название приложения: Поиск Изображений

Технология: Приложение разработано с использованием Angular

Основные характеристики приложения:

  • Приложение предназначено для поиска изображений. Полученные изображения выводятся на экране пользователя в виде карточек. Каждая карточка содержит предварительный просмотр изображения и дополнительную информацию, если доступна.
  • Пользователь может сохранять понравившиеся изображения. Сохраненные изображения хранятся на устройстве пользователя.
  • Приложение предоставляет вкладку "Сохраненные изображения", где пользователь может просматривать и управлять списком сохраненных изображений.
