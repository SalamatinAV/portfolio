Название приложения: Поиск Изображений

Технология: Приложение разработано с использованием Angular

Основные характеристики приложения:

  • Пользователь может вводить поисковый запрос, и приложение отправляет запрос на сервер для получения изображений, соответствующих заданному контексту.
  • Полученные изображения выводятся на экране пользователя в виде карточек. Каждая карточка содержит предварительный просмотр изображения и дополнительную информацию, если доступна.
  • Пользователь может сохранять понравившиеся изображения. Сохраненные изображения хранятся в local storage для долгосрочного доступа.
  • Приложение предоставляет вкладку "Сохраненные изображения", где пользователь может просматривать и управлять списком сохраненных изображений. Возможность удаления изображений добавляет гибкость.
  • При клике на изображение в карточке оно увеличивается в размере, предоставляя возможность более детального просмотра. Пользователь может листать изображения в увеличенном виде, используя кнопки "Назад" и "Вперед".
