Название приложения: Конвертер Валют

Технология: Приложение разработано с использованием Angular.

Основные характеристики приложения:

  • В верхней части страницы отображается текущий курс валюты USD и EUR по отношению к UAH. Эти данные автоматически обновляются для предоставления актуальной информации.
  • Также в верхней части страницы предусмотрены два роута: "Курс Гривны" и "Конвертер". По умолчанию пользователь находится на роуте "Курс Гривны".
  • Главный блок на роуте "Курс Гривны" отображает курсы иностранных валют по отношению к гривне. Эти данные также являются актуальными и обновляются в реальном времени.
  • Переход на роут "Конвертер" позволяет пользователю использовать инструмент конвертации валют. Здесь пользователь может выбрать валюту, указать сумму и выбрать целевую валюту для конвертации.
