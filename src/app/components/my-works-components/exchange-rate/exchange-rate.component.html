Название приложения: Конвертер Валют

Технология: Приложение разработано с использованием Angular.

Основные характеристики приложения:

  • В верхней части страницы отображается текущий курс валюты USD и EUR по отношению к UAH.
  • Также в верхней части страницы предусмотрены две вкладки : "Курс Гривны" и "Конвертер". По умолчанию пользователь находится на вкладке "Курс Гривны".
  • Главный блок на вкладке "Курс Гривны" отображает курсы иностранных валют по отношению к гривне.
  • Переход на вкладку "Конвертер" позволяет пользователю использовать инструмент конвертации валют. Здесь пользователь может выбрать валюту, указать сумму и выбрать целевую валюту для конвертации.
