Название приложения: Список Дел

Технология: Приложение разработано с использованием Angular.

Основные характеристики приложения:

  • Приложение предоставляет пользователю удобный способ организации задач. В основном окне размещены две колонки, где пользователь может видеть свой текущий список дел и список уже выполненных задач.
  • Пользователь может легко добавлять новые задачи, вводя название в соответствующее поле и нажимая кнопку "Добавить". Когда задача добавляется, она отображается в колонке "Список задач" с указанием названия задачи, даты и времени её добавления, а также кнопки для выполнения задачи.
  • После выполнения задачи пользователь может кликнуть на кнопку "Выполнено", и задача перемещается в колонку "Список Выполненных задач". При этом задача автоматически удаляется из колонки "Список задач". В колонке "Список Выполненных задач" также отображаются название задачи, дата и время выполнения, а также кнопка для удаления задачи из списка выполненных дел.
